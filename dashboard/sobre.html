o que é o astraea

O Astraea é um sistema de monitoramento que coleta, processa e apresenta dados sobre asteroides que passam perto da Terra e eventos solares. Os dados vêm diretamente das APIs públicas da NASA e são processados por um pipeline de dados automatizado.

Cada asteroide recebe uma classificação de risco calculada por um modelo de machine learning treinado com dados históricos. Os eventos solares — como ejeções de massa coronal (CMEs) e tempestades geomagnéticas (GSTs) — são monitorados e classificados por intensidade.

O objetivo é tornar dados científicos complexos acessíveis de forma clara e visual.
